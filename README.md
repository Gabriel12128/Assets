# 🎮 Game Assets

Repositório contendo assets utilizados em projetos de jogos, modelos, texturas e outros recursos gráficos.

---

## 📌 Sobre

Este repositório foi criado para armazenar e organizar assets utilizados em desenvolvimento de jogos e projetos pessoais.

Alguns assets podem ter sido criados manualmente, enquanto outros foram obtidos de fontes externas com suas respectivas licenças.
